📂 System Profile

Loopconomy is a modular Discord Kernel designed for educational exploration of JavaScript, Python, and PostgreSQL.

The bot features a multi-tiered trust system (Lvl 1-5) that allows for safe, sandboxed execution of community-developed addons.

🛠 Active Modules

  • Economy & Check-in System
  • PostgreSQL Currency Persistence
  • Multi-Deck 2-Shoe System
  • EUMI Hot-Reload Engine
  • Anti-Soviet Pipebomb Logic
  • FileSystem Note Encryption

🎲 Deployed Games

  • High-Stakes Poker (NLHE, PLO, STUD)
  • Blackjack (Casino Pro-Rules)
  • Mega Slots (Class-based Multipliers)
  • Rigged Coin Flip (10% Variance)
  • Russian Roulette (Mute Penalty) [WIP]
